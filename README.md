# Memory Puzzle Game

## Description
The Memory Puzzle Game is an engaging and challenging game designed to test your memory and concentration skills. Players are presented with a grid of cards, each hiding a unique image. The goal is to match pairs of images by flipping two cards at a time, trying to remember the positions of previously revealed cards.

## Setup Instructions
1. **Clone the repository**
   Run the following command in your terminal:
   ```bash
   git clone https://github.com/Eitezazur/Memory-puzzle-game.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Memory-puzzle-game
   ```

3. **Install dependencies**
   If the project uses npm, run:
   ```bash
   npm install
   ```
   Or if it uses yarn:
   ```bash
   yarn install
   ```

4. **Run the game**
   To start the game, use the following command:
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```

5. **Open your browser**
   Go to `http://localhost:3000` to start playing.

Enjoy the game!
